# webcheatsheets
Шпаргалки для всяких инструментов

## Структура сайта

* Главное - последние статьи
* Категории
* Редактор постов - для пользователей
* Админка

## Функции

* Отображение постов
    * Отображение всех постов
    * Отображение только выбранных категорий
* Редактор постов
* Комментарии
* Регистрация
* настройка профила - выбор интересующих категорий
